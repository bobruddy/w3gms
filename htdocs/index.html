This webserver is up to serve an ics file that contains all the calendar information for the w3gms repeater round table and workbench. You can subscribe to this calendar to stay up to date with the following URL: https://w3gms.o4x.net/w3gms.ics.

URL For information for this repeater is https://w3gmsrepeater.com/

If you want to see how this parse works and how it is run you can find the python script and Docker file at https://github.com/bobruddy/w3gms. If you have improvements feel free to submit a pull request, I may actually accept it.
